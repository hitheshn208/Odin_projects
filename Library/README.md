# Library

A simple library management app to keep track of your books.

## Live Demo

ðŸ”— [View the app](https://library-book-management-liard.vercel.app/)

## Features

- Add books with title, author, and category
- Mark books as read or unread
- Delete books from your library
- Color-coded categories
- Empty state when no books are added

## How to Use

1. Click "Add Book" button
2. Enter book details (title, author, category)
3. Click "Add" to save the book
4. Toggle read/unread status with the checkbox
5. Delete books using the delete button

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
