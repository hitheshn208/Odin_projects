# Tic-Tac-Toe

A classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features

- Two-player game mode
- Custom player names
- Choose between X and O symbols
- Score tracking
- Visual highlighting of winning combinations
- Reset button to start a new game

## How to Play

1. Enter player names when prompted
2. Player 1 chooses their symbol (X or O)
3. Players take turns clicking on empty cells
4. The first player to get three of their symbols in a row (horizontally, vertically, or diagonally) wins
5. Click the Reset button to start a new round

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
