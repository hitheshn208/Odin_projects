# Calculator

A fully functional calculator built using **HTML, CSS, and JavaScript** as part of **The Odin Project** curriculum.  
This project focuses on implementing **core calculation logic**, **state management**, and **user input handling** through both on-screen buttons and keyboard input.

---

## Project Overview

This calculator supports basic arithmetic operations with proper validation and error handling.  
It allows users to interact using either the **mouse** or **keyboard**, providing a smooth and intuitive experience.
