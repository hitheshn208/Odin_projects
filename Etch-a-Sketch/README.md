# Etch-a-Sketch

A browser-based drawing app inspired by the classic Etch-a-Sketch toy.

## Features

- Draw by clicking and dragging your mouse
- Change grid size (16x16 to 100x100)
- Random color mode for rainbow drawing
- Select specific colors with color picker
- Reset button to clear the canvas

## How to Use

1. Press and drag your mouse over the grid to draw
2. Use "Random Colors" for rainbow effect
3. Pick a specific color with the color picker
4. Change grid size by entering a number (16-100) and clicking "Change"
5. Click "Reset" to clear your drawing

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
