# Ichiraku Ramen Restaurant

A Naruto-themed restaurant website built with HTML, CSS, and JavaScript, bundled with Webpack.

ðŸ”— [Visit the restaurant](https://restraunt-page-p28ywqnxe-hithesh-s-projects-b3227f1a.vercel.app/)

## Features

- **Dynamic Tab Navigation** - Home, Menu, and Contact pages
- **Responsive Design** - Works on all device sizes
- **Webpack Bundled** - Modern build setup with asset management
- **Naruto Theme** - Inspired by the iconic Ichiraku Ramen from Naruto
- **Signature Ramens** - Classic Konoha, Deluxe Naruto, Hidden Leaf Tonkotsu, and Fire Dragon
- **Contact Form** - Get in touch with the restaurant

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Webpack
- Custom Fonts and Assets

## Development

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
